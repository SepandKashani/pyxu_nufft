Installation
------------

You can install ``pyxu_nufft`` via ``pip``:

.. code-block:: bash

   pip install pyxu_nufft


Developer Install
+++++++++++++++++

The HTML docs can be built using Sphinx:

.. code-block:: bash

   pip install pyxu_nufft[dev]
   pre-commit install

   tox run -e doc   # build docs
   tox run -e dist  # build packages for PyPI
